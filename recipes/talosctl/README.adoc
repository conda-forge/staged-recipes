= talosctl

A conda-forge recipe for https://github.com/siderolabs/talos[talosctl],
the CLI for managing Talos Linux.

== go-licenses --ignore flag

The `go-licenses save` build step uses `--ignore github.com/siderolabs/talos`
to exclude the talos module itself from license collection.
Without this flag, `go-licenses` copies the entire talos source tree
(1.3M+ files, ~37 GB) into the `library-licenses/` output directory,
because the module root contains the full monorepo.
The top-level `LICENSE` file already covers the project itself,
so only third-party dependency licenses need to be collected.
